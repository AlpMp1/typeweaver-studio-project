TypeWeaver Studio 是一个 AI 类型迁移与代码质量修复工作台，用来把历史 JavaScript 项目逐步迁移到 TypeScript。 它会读取仓库结构、运行时日志、隐式数据模型、测试用例与错误栈，自动推断类型、生成迁移计划、补齐类型定义与回归测试，并输出可审查的 patch 报告。

Designed for gradual type migration.

TypeWeaver 不会一次性重写整个项目，而是按“边界识别、类型推断、最小迁移、测试补齐、风险审查”的方式， 帮助团队把历史 JavaScript 项目安全迁移到 TypeScript。
